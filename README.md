Glow Notebooks

This is a simple product landing page created to practice HTML and CSS.  
The project showcases a clean layout, custom styling, and a product presentation for a fictional notebook brand called “Glow Notebooks”.

Features:

- Clean and modern product layout
- Custom CSS styling
- Responsive design structure
- Product images included
- Simple, beginner-friendly code
- Organized project folders (HTML, CSS, images)

Technologies used:

- HTML5
- CSS3

Purpose:

This project was created to practice building product landing pages and organizing web assets for small front-end projects.  
It is part of my portfolio to demonstrate basic front-end development skills and UI layout structure.
